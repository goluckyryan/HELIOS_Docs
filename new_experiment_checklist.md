# new_experiment_checklist.md — New Experiment Setup (MANDATORY)

When starting analysis on any new experiment, create a subdir under `~/digios_11C_2/analysis/working_Helios/<expName>/`.

---

## Required Files

| File | Content |
|------|---------|
| `exp.md` | Run log, array configs, Ex ranges per config, good runs, RDT cut files, beam notes |
| `<exp>_analysis_procedure.md` | Full calibration pipeline with lessons learned |
| `reactionConfig.txt` | Beam/target/recoil definition |
| `Ex.txt` | Excitation energies to simulate |
| `detectorGeo_<config>.txt` | One per array position |
| `detectorGeo.txt` | Symlink → active config |
| `reaction_<config>.dat` | Generated by Transfer per geometry |
| `reaction.dat` | Symlink → active config |
| `transfer_<config>.root` | Generated by Transfer binary (`make` in `Cleopatra/`) |
| `transfer.root` | Symlink → active config |
| `correction_e_alpha.dat` | Energy cal (generated fresh — NOT copied from Ryan) |
| `correction_e.dat` | Symlink → correction_e_alpha.dat |
| `correction_xf_xn.dat` | Xf/Xn gain match |
| `correction_xfxn_e.dat` | Xs→E linearity |
| `correction_scaleX.dat` | X position scale |
| `alpha_calib_228Th_v5.C` | Calibration macro |
| `xfxn_calib.C` | xnCorr macro |
| `xfxne_calib.C` | xfxne macro |
| `xscale_calib.C` | xScale macro |
| `rdtCuts_<name>.root` | RDT cut file (copy, not symlink) |
| `GeneralSortMapping.h` | Channel mapping (copy) |

---

## exp.md Must Include

- Array position configs (position, Ex range focus, which runs)
- Full run log (start/stop, duration, events, beam quality, flags)
- Physics states (Ex.txt entries, which config they target)
- Monitors.C settings per config (exRange, rdtCutFile)
- Recommended runs for physics analysis

---

## Rules

- `~/digios_11C_2/analysis/working_Helios/<expName>/` is fully decoupled from `working/`
- Config files: **real copies** (not symlinks to working/)
- Code files (Monitors.C, Armory/): symlinks OK
- Data files (root_data/): symlink OK
- Run full 4-step calibration from scratch — see `calibration_notes.md`
- No copying Ryan's params — generate everything independently

---

## Calibration Order (non-negotiable)

1. Energy → `correction_e_alpha.dat`
2. Xf/Xn gain → `correction_xf_xn.dat`
3. Xs→E linearity → `correction_xfxn_e.dat`
4. X scale → `correction_scaleX.dat`

---

## Plot Index Setup (do this first)

Create `plot_index.md` in the experiment working dir before generating any plots:

```markdown
# Plot Index — <expName>

| Plot | File | Description |
|------|------|-------------|
| Plot-001 | ... | ... |
```

- Format: `Plot-NNN` (zero-padded, sequential, never reused)
- Every plot filename and title must include the index number
- See `HELIOS_Analysis_Workflow.md` → Plot Indexing Convention
